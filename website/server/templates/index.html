Machine Learning Scoring Predictions and The Sportsbook Market

What is the Best Way to Predict Future NBA Scores Based on Past Data?


The purpose of this project is to identify the game-level variables that have the most influence on team scoring, create a machine learning model based on a team's past performance in these selected variables to predict individual game scores and evaluate the success of these predictions based on the decisions that would be made given the available data and market conditions.

This project involves all three phases of the ETL Life Cycle. Data Extraction of historical team stats from Basketball Reference and historical market lines from OddsShark was handled with the Requests and Selenium Python Libraries. Data Cleaning and Transformation was handled with the Pandas, NumPy and Regex Libraries. Model Creation was handled with Sci-Kit Learn Libraries. Model Evaluation was handled by my own custom defined ModelAnalyzer Class which transformed single score point estimates into their implied market counterparts in order to easily compare traditional sportsbook markets with my sole point estimate for each game.

Data Storage is handled by SQL database on the website's backend. Data Loading is handled by the Python Flask and SqlAlchemy libraries and SQL queries to the cleaned databasee. Data Presentation via this Web Application is handled by the Javascript Libraries: Plotly, D3, jQuery, Popper, Observable, Color-Thief and built on the Bootstrap 4 Framework.
